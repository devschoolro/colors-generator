Propunere de proiect:

1. Se genereaza 3 coduri de culoare (RGB sau HEX)
2. Se afiseaza 1 cod de culoare
3. Se afiseaza cele 3 culori corespunzatoare codurilor de la punctul (1)
4. Utilizatorul trebuie sa ghiceasca ce culoare
corespunde codului afisat

Exemplu:

1. Coduri generate

rgb(0,0,0) -> negru
rgb(255,255,255) -> alb
rgb(255,0, 0) -> rosu

2. Se afiseaza codul rgb(0,0,0)

3. Se afiseaza culorile: alb, rosu, negru

4. Utilizatorul trebuie sa aleaga negru pentru a raspunde corect.
